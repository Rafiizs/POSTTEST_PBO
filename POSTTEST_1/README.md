# Deskripsi Program

Program ini merupakan aplikasi sederhana berbasis Java yang dibuat menggunakan konsep **Pemrograman Berorientasi Objek (OOP)**.

Program ini berfungsi untuk mengelola daftar manga yang tersedia.
User dapat:

* Menambahkan manga
* Melihat daftar manga
* Menghapus manga

Setiap manga memiliki atribut:

* ID Manga
* Judul Manga
* Author
* Harga

Program dibuat menggunakan konsep:

* Class
* Object
* ArrayList
* Method

Materi yang digunakan sesuai dengan **Modul 1 dan Modul 2**.

---

# Struktur Program

Program terdiri dari beberapa class:

### 1. Main.java

Berfungsi sebagai program utama yang menjalankan menu dan interaksi dengan user.

### 2. Manga.java

Berisi class Manga yang memiliki atribut:

* id
* judul
* author
* harga

Class ini juga memiliki constructor dan method getter.
